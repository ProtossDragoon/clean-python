     1.   키워드: `__dict__`, `__set__`, `__get__`

     1.   키워드: `__dict__`
     2.   용어: 훅(hook), 오브젝트 훅
     3.   용어: 인스턴스 애트리뷰트 딕셔너리
48.  메타클래스 대신 `__init_subclass__` 를 사용해 하위 클래스를 검증하라.
